Beat Blast - Music Player

Beat Blast is a simple and efficient music player built by students under the leadership of Aditya Singh. It is designed to provide an enjoyable music experience with basic features.
Features

    Easy navigation and user interface
    Play, pause, skip and rewind tracks
    Create and manage playlists
    Shuffle and repeat tracks
    Supports various audio formats such as MP3, WAV, and AAC

Installation

    Clone the repository to your local machine using git clone https://github.com/ZyntaxZone/Beat-Blast.git
    Open the project in your preferred code editor
    Run the index.html file in your browser

Usage

    Add your music files to the "Music" folder
    Open the Beat Blast application in your browser
    Click the "Add Music" button to import your music files
    Use the player controls to play, pause, skip and rewind tracks
    Create and manage playlists as per your preferences
